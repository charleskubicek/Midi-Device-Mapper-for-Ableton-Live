Review code that is uncommited and in the last commit.

Review this work thoroughly, including the tests.

- Ensure it's technically sound and of minimal complexity.
- Report duplication (not all duplication is bad)
- Seek opportunities for simplifying, even it if many change product behaviour.
- Ensure each test, new and existing (existing related to the work done) earns it's keep; tell me if some seem overly simplistic or are now unnecessary
- Highlight classes, function and artefacts that have grown too big

Present all points to me first.
